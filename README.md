## Scripts

- ### Installing dependencies on all packages

  ```bash
  $ yarn run install:all
  ```

- ### Running ESLint on all packages

  ```bash
  # basic
  $ yarn run lint

  # run and fix all errors
  $ yarn run lint:fix
  ```

- ### Running tests on all packages

  ```bash
  # unit tests
  $ yarn run test

  # integration tests
  $ yarn run test:int

  # e2e tests
  $ yarn run test:e2e

  # all tests
  $ yarn run test:all

  # test coverage
  $ yarn run test:cov[:int|:e2e|:all]
  ```
